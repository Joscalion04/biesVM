Class: FunDefContext

FunDefContext()

new FunDefContext()

Clase que extiende `antlr4.ParserRuleContext` y define el contexto para la definición de funciones en el lenguaje BIES. Esta clase representa un bloque de código que define una función, incluyendo su nombre, parámetros y cuerpo.
